# 📊 UnitPilot

**UnitPilot** is a powerful smart assistant designed for unit economics modeling and product analytics directly inside your spreadsheets. The script transforms a regular spreadsheet into a full-scale analytical workspace, automating complex mathematical calculations, forecasting, and statistical tests.

---

## 🚀 Key Features

### 📈 Unit Economics & Metrics
* **Automated Metric Calculation:** Instant calculations for Revenue, CAC (Customer Acquisition Cost), LTV (Lifetime Value), ROMI (Return on Marketing Investment), ARPU/ARPPU, and profit margins.
* **Quick Formulas:** Copy and immediately paste ready-to-use formulas into your target cells.

### 🧪 Statistics & A/B Testing
* **Statistical Tests:** Run A/B, A/B/n testing, t-test, Analysis of Variance (ANOVA), and correlation analysis.
* **Traffic Splitting:** Quasi-random user distribution across groups with 6 different allocation types.

### 🔮 Forecasting & ML
* **Advanced Forecasting:** Time-series modeling (ARIMA), Monte Carlo simulations, and classic linear forecasting.
* **Machine Learning:** K-Means clustering for smart customer and order segmentation.

### 🔬 Data Quality & Cohort Analysis
* **Data Validation:** Automatic detection of missing values, anomalies (outliers), and data normality checks.
* **Cohort Analysis:** Generate user retention tables (Retention Rate) tracked by months or weeks.

### 🎨 Visualization & Templates
* **Interactive Dashboards:** Auto-generation of visual charts, heatmaps, and conversion funnels.
* **Ready-made Business Templates:** Pre-built marketing, product, and financial dashboards.

---

## 🌍 Multilingual UI & Learning

* **Interface:** Full native support for three languages — **English, German (Deutsch), and Russian**.
* **Knowledge Base:** Built-in assistant capable of answering metric methodology questions in any of the supported languages.

---

## 🛠️ Installation & Setup

1. Open your Google Sheet.
2. In the top menu, go to: **Extensions** -> **Apps Script**.
3. Copy the code from the `main.js` file in this repository and paste it into the script editor.
4. Save the project and refresh your spreadsheet page.
5. A new menu item named **UnitPilot** will appear, allowing you to launch all the tools.
